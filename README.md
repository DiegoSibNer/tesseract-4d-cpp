# Tesseract 4D Visualizer - C++
Motor gráfico que modela y anima un hipercubo en tiempo real aplicando álgebra lineal.

## Tecnologías
- C++
- olcPixelGameEngine
- Visual Studio

## Características
- 16 vértices y 32 aristas en espacio 4D
- 6 rotaciones simultáneas en planos 4D (XY, XZ, XW, YZ, YW, ZW)
- Proyección sucesiva 4D → 3D → 2D
- Animación fluida dependiente del tiempo real

## Cómo ejecutar
1. Clona el repositorio
2. Abre el proyecto en Visual Studio
3. Compila y ejecuta
